# VibeLearning - 基于Vue.js的课件交互式学习页面

## 快速上手

```
git clone git@github.com:404MaximWang/VibeLearning.git

cd VibeLearing

npm ci

npm run dev
```

本项目暂不支持纯文本LLM。请使用OpenAI Compatible API并使用多模态模型。
